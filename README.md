# 柔度算子法

提出一种基于柔度算子的动力学建模方法，利用神经网络预测柔性体的柔度张量场，建立外力场与位移场之间的算子映射。该方法省去了有限元计算中最繁琐耗时的网格划分与全局方程求解过程，预测得到的柔度张量可直接嵌入现有的动力学与接触仿真框架中，保持良好的物理一致性与可解释性。在保持精度的前提下，计算效率相比有限元方法有数量级提升。
