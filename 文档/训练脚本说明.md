# 模型训练脚本说明 (`train_model.py`)

该文件负责神经网络模型的训练流程控制，包括：

- 数据加载  
- 损失函数定义  
- 优化器与学习率调度策略配置  
- 训练过程中的模型保存与早停逻辑  

同时兼容GPU，CPU，MPS训练。

---

## 核心组成

### 1. `MyData` 数据集封装类

- 对输入与标签进行张量化封装，供 `DataLoader` 使用  
- 支持批量加载与多进程读取  

### 2. 损失函数族（`loss_fn` / `loss_fn_rel` / `loss_hybrid` 等）

- 默认使用 **MSE** 损失  
- 预留相对误差、对数域误差及对称性约束等实验性损失函数  
- 便于针对不同物理量与数值尺度切换训练策略  

### 3. `train()` 主训练函数

负责完整训练流程，包括：

- **设备选择**：CUDA / MPS / CPU  
- **数据集与 DataLoader 构建**  
- **模型实例化**：MLP 结构，支持多 block 与可选 dropout  
- **AdamW 优化器与参数分组**：区分核参数与其余参数  
- **余弦退火 + warmup 学习率策略**  
- **AMP 自动混合精度训练**（float16）  
- **Early Stopping** 与周期性模型保存  

---

## 关键参数说明

| 参数 | 作用 |
|----|----|
| `hidden_dim` / `block_num` | 控制网络宽度与深度 |
| `batchsize` / `numworks` | 控制训练吞吐与数据加载并行度 |
| `max_lr` / `lr_min` / `use_cosine` | 学习率与调度策略 |
| `use_dropout` / `use_L2` | 正则化相关开关 |
| `epoch_num` / `patience` | 训练轮数与早停策略 |

---

## 训练输出

- 定期保存模型：`mlp_model_epoch*.pth`  
- 最优模型保存为：`mlp_model_best.pth`  
- 自动记录训练参数与耗时至 `params.json`  
- 输出训练与测试损失曲线，用于收敛性分析
